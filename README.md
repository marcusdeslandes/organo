# 🌐 Organo | Gerenciador de Organogramas Dinâmicos

![React](https://img.shields.io/badge/React-20232A?style=for-the-badge&logo=react&logoColor=61DAFB)

> Uma aplicação interativa e moderna desenvolvida em React para criação, visualização e gerenciamento de organogramas corporativos de forma simples e intuitiva.

[🔗 Clique aqui para testar o projeto rodando ao vivo!](https://organo-puce.vercel.app/)

---

## 🚀 Funcionalidades

*   ➕ **Adicionar Colaboradores:** Cadastre nome, cargo, imagem e escolha o time.
*   🎨 **Organização por Cores:** Times separados visualmente para facilitar a leitura.
*   📱 **Responsivo:** Funciona bem no computador ou no celular.

---

## 🛠️ Tecnologias

*   **React** (Componentização e Estados)
*   **CSS3** (Estilização)
